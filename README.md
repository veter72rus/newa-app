# נ.ע.א — השוואת תשואות מסלולי ביטוח

Yield comparison tool for Israeli insurance investment tracks.

## Stack

- **Next.js 15** (App Router) + TypeScript
- **Tailwind CSS v4**
- **PostgreSQL** (Prisma ORM — coming soon)
- **Vercel** deployment

## Hebrew / RTL

- `<html lang="he" dir="rtl">` set in root layout
- `Content-Language: he` meta tag
- RTL-aware Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
